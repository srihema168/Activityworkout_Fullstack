# **Active Fitness**

**Active Fitness** is a web application designed to help users plan, track, and optimize their fitness journey with features like workout tracking, customizable plans, and progress analytics.

---

## **Features**

- Log and monitor workouts.  
- Create personalized workout plans.  
- Access an exercise library with instructions and videos.  
- Visualize progress with charts and analytics.  
- Connect with a fitness-focused community.  

---

## **Technologies Used**

- **Frontend:** React.js  
- **Backend:** Node.js with Express.js  
- **Database:** MongoDB
